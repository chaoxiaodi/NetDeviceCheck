NetDeviceCheck

# 项目说明
通过python netmiko模块实现H3C网络设备巡检<br />
分析输出的信息并进行可视化输出<br />
cmdInfo 包含所有需要在设备执行的命令<br />
deviceInfo.py 包含所有需要巡检的网络设备信息<br />

|设备名称 | 设备IP | 用户名 | 密码 |
|---------|------------|------|---------|
|'sw0013 |192.168.1.13 |admin |12345678'|
<br />
ps：设备需要开启SSH登录并允许列表里的用户登录<br />
